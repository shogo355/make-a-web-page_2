1.1 The Basics of the Web and HTML
"Computers are stupid because
they cannot interpret mistakes"

Basic definitions

Web...The way computers communicate by using http protocol.
http..."Hypertext Transfer Protocol" The protocol used to transmission and reception of HTML files between Web browser and Web server.
HTML..."Hyper Text Markup Language" A language that is used for constract the web page.

How HTML works

HTML documents use "tags" to what to show on the browser.
There are two types of ELEMENTS described by "tags".

Type1:BLOCK elements

Block elements normally start and end with a new line which means the browser will display line breaks before and after the block element. They can be described as below.
<OPENING TAG>
   <CONTENTS>
</CLOSING TAG>
Opening tags and Closing tags have exactly the same name except closing tags have "/"at the beginning of the tags.
Examples of block elements are described as follows.
<div>
   <CONTENTS>
</div>
<h1>
   <HEADLINE>
</h1>

Type2:INLINE elements

The INLINE elements are normally displayed without line breaks. They can be described as follows.
<TAG><contents>
<OPENING TAG><contents></CLOSING TAG>
Examples of inline elements are described as follows.
<br>:Begin a new line
<b>content</b>:Make contents bolded
<em>content</em>:Make contents Italic

HTML attributes

Some of the tags involves HTML attributes in their tags.
In case of block elements, HTML attributes looks like this.
<TAG ATTR="value">conetnts</TAGS>
For inline elements, HTML attributes looks like this.
<TAG ATTR="value">
Example of the HTML attributes are follows.
<a href="www.udacity.com">Udacity</a>:Create hyperlinks as Udacity
<img src="url of the image" alt"text">:Display the linked image.

1.3 Adding CSS style and HTML structure
"Avoid repetition is essential in all areas of programming"

　

What does CSS do?Why do we need it?

CSS stands for Cascading Style Sheets.CSS include style properties for HTML elements. The each styles have class name. By giving the class name to HTML elements, the elements can refer the style.
CSS is important since it can avoid repetition. Using class can avoid describing same properties again and again. Avoiding repetition is not just easier for us to write HTML but also it can avoid errors. If you did not avoid repetition and repeat same code many times, once you see the error in the code you must change all of the code. However, if you use CSS to avoid repetition, you only need to change once.

How to add CSS style

There are three ways to include CSS in HTML. All of the method must include following code before using CSS.
<!DOCTYPE html>
   <html>
      <head>
      </head>
            <body>
            </body>
   </html>
Between <html> and </html> is the area where HTML code written.
Between <head> and </head> is the area where information like CSS can be included as explain later.

1. Write CSS in <head> of the HTML

<head>
   <style>
      div{

2. Link your HTML to a separate CSS file

<head>
   <link rel="stylesheet" href="main.css">

3.Write your style inline with your HTML

<body>
   <div style="background-color:red;">

The structure of the CSS style sheet

To add a "style" write
.selector{
   attribute:value;
}
For "selector", write the title of the style you will describe.
For "attribute", write the property of the style, such as background-color, border-style, etc.
For "value", write the values for given attribute. For example, if the attribute is about color, write "red".
On CSS, "attribute:value;" is collectively called "declaration".

How to apply CSS to HTML

Once you declared your selector, you can apply it to HTML by using class. The example is given below.
<div class="selector">
</div>
